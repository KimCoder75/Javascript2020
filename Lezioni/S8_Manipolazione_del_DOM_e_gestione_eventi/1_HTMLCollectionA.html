L'oggetto HTMLCollection

Un oggetto HTMLCollection è un elenco di elementi HTML simile a un array.

Metodi come getElementsByTagName() restituiscono una HTMLCollection.

Proprietà e metodi:

Le seguenti proprietà e metodi possono essere utilizzati su un oggetto HTMLCollection:

  • item() Restituisce l'elemento all'indice specificato in una HTMLCollection
  • length() Restituisce il numero di elementi in una collezione HTMLCollection
  • namedItem() Restituisce l'elemento con l'ID, o il nome specificato, in una collezione HTMLCollection

Esempio

Uso dell metodo getElementsByTagName() per restituire una HTMLCollection di tutti gli elementi p contenuti in document ed
il numero di elementi contenuti nella HTMLCollection

<script>
let x = document.getElementsByTagName("P");
document.write("I tag <p> contenuti fanno parte di :"+x+
"<br>"+"il loro numero è: "+x.length+"<br>"+"Il testo contenuto nel primo elemento accessibile tramite item(0).innerText è: "+
"<br>"+x.item(0).innerText+"+x.length+"<br>"+"Il testo contenuto nel secondo elemento accessibile tramite item(1).innerText è: "+
"<br>"+x.item(1).innerText);
</script>

Nota Bene: i "<br>" sono necessari solo se lo script è incorporatoto nella pagina HTML, come in questo caso,
qualora fossere scritti in un file javascript esterno al documento, per andarare a capo si userebbe \n, in questo modo:

let x = document.getElementsByTagName("P");
document.write("I tag <p> contenuti fanno parte di :"+x+\n
"il loro numero è: "+x.length+\n+
"Il testo contenuto nel primo elemento accessibile tramite item(0).innerText è: "+\n
x.item(0).innerText+\n
"Il testo contenuto nel secondo elemento accessibile tramite item(1).innerText è: "+\n
x.item(1).innerText);
